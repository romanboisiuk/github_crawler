# GitHub Parser

## Requirements

- Poetry for Python dependency management

### Setting Up Your Environment

1. Install Poetry:
   ```bash
   pip install poetry
   ```

2. Install dependencies:
   ```bash
   poetry install
   ```
   
3. Run the crawler
   ```bash
   python parser.py
   ```

4. Run the tests
   ```bash
   python -m unittest -v tests.py 
   ```
